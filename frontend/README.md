# IndiaMART AI RFQ System - Frontend Demo

Interactive demo showcasing the AI-powered Intelligent RFQ System for IndiaMART.

## Tech Stack
- Next.js 14 (Static Export)
- Tailwind CSS (IndiaMART design tokens)
- Framer Motion (animations)
- Lucide React (icons)
- TypeScript

## Deploy to Vercel

[![Deploy with Vercel](https://vercel.com/button)](https://vercel.com/new/clone?repository-url=https://github.com/thegauravmahto/indiamart-rfq-system&root-directory=frontend)

## Local Development

```bash
cd frontend
npm install
npm run dev
```

Open http://localhost:3000

## Features
- IndiaMART-themed UI with exact brand colors (#2E3192, #007A6E, #00A699)
- Interactive chat demo simulating the AI RFQ flow
- Before/After comparison showing the transformation
- Architecture visualization of the 3-agent pipeline
- Impact metrics dashboard
- Fully responsive, works on mobile

## Build
```bash
npm run build  # Creates static export in /out directory
```
